=======================
Reports on DHI website
=======================

.. note:: Currently, website reports can be accessed by logging in and visiting the Settings page on your account profile.

All reports can be exported to CSV for further manipulation.

Debt usage report
==================
The debt usage report provides an overview of debts created on the platform and a way for content managers to access the summary results provided to the website user.

The report includes:

* Problem profile ID
* Debt problem ID
* Debt entity ID
* User ID
* Debt type
* Problem type
* Creditor name
* Debt name
* Debt created date
* Debt changed date
* Link to summary

Prioritized debt report
========================

This report is useful to see how many users are prioritizing debts and what happens once they do - do the debt types stay prioritized or does the user complete them?

This report shows:

* Debt problem ID (for each prioritization, there will be 1 problem ID with one or more debt IDs)
* Debt ID
* Profile ID
* Debt type
* Date created
* Date last changed
* Status

  * Prioritized
  * In progress
  * Archived


All debt details report
==========================

This report shows details across all debts in the system. This incudes:

* Debt ID
* Debt problem ID
* Profile ID
* UID, if the user has an account
* Amount
* Debt type
* Last payment date
* Creditor name
* Creditor type
* Is debt collector (yes or no)
* Debt status
* Date created
* Date changed

Debts by location
===================

This report shows where debts are located throughout the state, including the city, county, zip code, and zip code suffix. The report includes:

* Debt type
* Problem type
* City
* County
* Zip code
* Zip suffix
* Created date
* Last changed date
* Profile ID
* Debt problem ID
* Debt entity ID

Debt profile metadata
=======================

This report includes all of the profile metadata attached to specific debts. For example, which debts involve debt buyers or are in small claims.

The report includes:

* Debt entity ID
* Debt type
* Problem type
* Debt metadata
* Debt created date
* Debt last changed date
* Problem profile ID

Problem profile metadata
=======================

This report includes all of the profile metadata attached to specific problem profiles. ** Problem metadata is less common than debt metadata**

The report includes:

* Profile metadata
* Profile created date
* Profile last changed date
* Problem profile ID
