=======================
Reports on DHI website
=======================

.. note:: Currently, website reports can be accessed by logging in and visiting the Settings page on your account profile.

All reports can be exported to CSV for further manipulation.

Prioritized debt report
========================

This report is useful to see how many users are prioritizing debts and what happens once they do - do the debt types stay prioritized or does the user complete them?

This report shows:

* Debt problem ID (for each prioritization, there will be 1 problem ID with one or more debt IDs)
* Debt ID
* Profile ID
* Debt type
* Date created
* Date last changed
* Status

  * Prioritized
  * In progress
  * Archived


All debt details report
==========================

This report shows details across all debts in the system. This incudes:

* Debt ID
* Debt problem ID
* Profile ID
* UID, if the user has an account
* Amount
* Debt type
* Last payment date
* Creditor name
* Creditor type
* Is debt collector (yes or no)
* Debt status
* Date created
* Date changed
